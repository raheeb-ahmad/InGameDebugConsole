# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-20

### Added
- On-device logcat reading for Android runtime logs
- Unity log capture (Debug.Log, warnings, errors)
- Type filters to show/hide logs, warnings, and errors independently
- Live search to filter log entries by keyword
- Auto-scroll to keep the latest log entry in view
- Auto-show on error to surface the console when an error occurs
- UPM install support via Git URL
